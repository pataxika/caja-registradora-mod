# 🛒 Caja Registradora Pro - Lite

Una aplicación de terminal de punto de venta (POS) simplificada, construida íntegramente con **HTML5, CSS3 y JavaScript vanilla**. Ideal para pequeños comercios o como base para proyectos de gestión de inventario.

## ✨ Características

- **Gestión de Stock Real**: Los productos descuentan existencias automáticamente al agregarse al carrito.
- **Buscador en Tiempo Real**: Filtra productos por nombre instantáneamente.
- **Sistema de Pagos**: Selección de medios de pago (Efectivo, Débito, Crédito) con validación de confirmación.
- **Interfaz Responsiva**: Diseño moderno (Dark Mode) que se adapta a diferentes tamaños de pantalla.
- **Sin Dependencias**: Todo el código reside en un único archivo ejecutable sin necesidad de servidores o librerías externas.

## 🚀 Instalación y Uso

Este proyecto no requiere instalación de paquetes adicionales.

1.  **Clona o descarga** el archivo `index.html`.
2.  **Abre el archivo** en tu navegador favorito o usa la extensión **Live Server** en VS Code para una mejor experiencia.
3.  ¡Empieza a gestionar tus ventas!

## 🛠️ Tecnologías Utilizadas

- **HTML5**: Estructura semántica.
- **CSS3**: Layout con Grid y Flexbox, variables CSS para personalización de colores.
- **JavaScript (ES6)**: Lógica de estado para el carrito, filtrado de datos y manipulación del DOM.

## 📝 Instrucciones de Uso

1.  **Buscar**: Escribe el nombre de un producto en la barra superior.
2.  **Agregar**: Haz clic en la tarjeta de un producto con stock disponible.
3.  **Seleccionar Pago**: Elige un método en el panel derecho.
4.  **Confirmar**: Presiona el botón verde de "Confirmar Pago" para finalizar la transacción.
